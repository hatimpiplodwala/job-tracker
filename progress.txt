Phase 4 — Stack Migration                                           [done]
--------------------------------------------------------------------------
Replaced v1 Streamlit app with Next.js 14 (App Router) + FastAPI + Supabase.
Auth via @supabase/ssr (cookie-based, SSR-safe). CRUD endpoints with RLS.
Table with sort/filter/search, modal dialogs (add/edit/delete + duplicate
warning), CSV export, mobile-responsive layout. Deployed to Vercel + Render.

Security + perf hardening (4.8):
- job_url validation: http/https only, frontend regex + Pydantic field_validator
- CSV formula injection guard (CWE-1236) in lib/csv.ts
- Supabase auth.get_user cached in-process (60s TTL, 1024 cap, sha256 key)
- Skip duplicate-check round-trip when company/role unchanged on edit
- Settings reads env in __init__ (not at class-definition time)
